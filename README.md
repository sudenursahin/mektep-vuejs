# Mektep - Ottoman-Era Inspired E-Learning Platform

What if Udemy existed in the Ottoman Empire? This question inspired Mektep, a unique educational platform that combines modern web technologies with classical Ottoman aesthetics and educational principles.

## Technical Stack

- Vue.js 3 with Composition API
- Vite for development and builds
- Firebase (Authentication & Firestore)
- Pinia for state management
- Vuetify 3 with Ottoman-inspired customization
- Custom CSS with Ottoman design system

## Design Features

- Traditional Ottoman color palette (deep burgundy, gold accents)
- Classical Islamic geometric patterns
- Custom animations mimicking manuscript page turns
- Typography inspired by Ottoman manuscripts using Amiri font family

## Key Features

- User authentication and authorization
- Course management system
- PDF parsing with Arabic/Ottoman text support
- Real-time data synchronization
- Responsive layouts
- Optimized image loading and caching
